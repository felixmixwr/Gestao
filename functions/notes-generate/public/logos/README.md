# Logos das Empresas

Este diretório deve conter as logos das empresas para inserção nas notas fiscais.

## Arquivos necessários:
- `felixmix.png` - Logo da empresa Félix Mix
- `worldrental.png` - Logo da empresa World Rental

## Especificações:
- Formato: PNG (recomendado) ou JPG
- Tamanho: Máximo 200x100 pixels (será redimensionado automaticamente)
- Fundo: Transparente (PNG) ou branco
- Qualidade: Alta resolução para impressão

## Como usar:
1. Adicione as logos neste diretório com os nomes exatos
2. A função automaticamente encontrará e usará a logo baseada no parâmetro `company_logo`
3. A logo será inserida no lugar do placeholder `{{LOGO}}` no template Excel
